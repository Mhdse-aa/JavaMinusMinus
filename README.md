# Lexical Analyzer

# This project implements the **Lexical Analyzer** (scanner) for the simplified programming language **Java--**.  
The goal is to identify and classify all **tokens** from a given source file according to the provided grammar.

# Token Categories
| Category | Examples |

| **Identifiers** | `Main`, `myVar`, `sum_total` |
| **Keywords** | `class`, `if`, `else`, `while`, `return`, `public`, `int`, ... |
| **Operators** | `+`, `-`, `*`, `/`, `%`, `==`, `!=`, `<=`, `>=`, `&&`, `||`, `=`, `.length` |
| **Delimiters** | `{`, `}`, `(`, `)`, `[`, `]`, `;`, `,`, `.` |
| **Numbers** | `0`, `123`, `45_67`, `9L` |
| **String literals** | `"Hello World!"`, `"abc\\n123"` |
| **Character literals** | `'a'`, `'\\n'` |
| **Boolean literals** | `true`, `false` |
| **Null literal** | `null` |

# Requirements
Python **3.8+**

# Run the Lexer
python main.py
